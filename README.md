# acrux-patchset
================

Linux kernel v7.0 patches including additional features:

- Gentoo Base, Extra, and Experimental patches v7.0-1 (Gentoo linux-patches [infopage](http://dev.gentoo.org/~mpagano/genpatches/) or [repository](https://gitweb.gentoo.org/proj/linux-patches.git))

These patches can be applied directly on top of the vanilla Linux kernel version [7.0.1](https://cdn.kernel.org/pub/linux/kernel/v7.x/linux-7.0.1.tar.xz).
